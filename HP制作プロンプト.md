# Hi5 Tokyo — HP制作プロンプト

---

## 依頼概要

以下の仕様・コンテンツをもとに、Baseball5チーム「Hi5 Tokyo」の公式ホームページを HTML / CSS / JavaScript（単一ファイル）で作成してください。

---

## デザイン要件

- **テーマ**：明るく・元気・親しみやすい。暗い配色は使わないこと
- **メインカラー**：ピンク（#E0508A）
- **アクセントカラー**：ゴールド（#C9A44A）
- **背景**：白（#FFFFFF）・薄ピンク（#FFF0F7）を交互に使用
- **テキスト**：ダークチャコール（#1E1520）
- **フォント**：Google Fonts — Noto Sans JP（日本語） + Oswald（英字見出し）
- **レイアウト**：モバイルファースト・スマホ完全対応
- **ナビゲーション**：固定ヘッダー・スクロール時に縮小・スマホはハンバーガーメニュー
- **アニメーション**：スクロール時にフェードインで要素を表示（IntersectionObserver使用）
- **ターゲット**：経営者（スポンサー・パートナー候補）とプレイヤー（参加希望者）の2軸

## 技術仕様

- **ファイル構成**：index.html 1ファイルに HTML・CSS・JS をすべてインライン記述
- **外部依存**：Google Fonts のみ（CDN）
- **JS**：バニラJS のみ（jQuery・フレームワーク不使用）
- **スクロールアニメーション**：IntersectionObserver でフェードイン
- **フォーム**：バックエンド不要。送信時にJSでサンクス表示のみ
- **画像**：実写真がないセクションはピンク系グラデーションのCSS背景プレースホルダー（`<img>` タグに `src` 属性を入れて後から差し替え可能な形）
- **アクセシビリティ**：alt属性・aria-label・tabindex を適切に設定

---

## 注意事項

- 提案資料（スライドデッキ）のような堅い構成にしない。共感・温度感・人間味を重視すること
- 「経営者がスポンサーになりたい」「プレイヤーが参加したい」と思える設計
- テキストは読みやすく簡潔に。長文の羅列にならないよう注意
- モバイルで見たときに情報が詰まりすぎないよう余白を十分に取ること
- 写真プレースホルダーは後から実写真に差し替えやすいよう、クラス名やコメントを明記すること
